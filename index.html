5G Power Consumption Modelling

An interactive educational tool for comparing power consumption across D-RAN, C-RAN, and Cloud C-RAN architectures.

This project presents a software application for 5G NR Power Modeling developed in MATLAB. The tool includes an interactive GUI enabling users to explore and compare different 5G network architectures.

The software is distributed as a standalone executable file, making it accessible without a MATLAB license.
